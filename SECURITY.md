# Security Policy

## Reporting a Vulnerability

Please report vulnerabilities **privately** via GitHub's advisory system:

**[Report a vulnerability](https://github.com/gary23w/garrettstimpson.ca/security/advisories/new)**

Do **not** open a public issue. I'll acknowledge within 48 hours.

## Scope

This repo hosts a static Jekyll site (garrettstimpson.ca).
Attack surface: GitHub Pages deployment, Actions workflow, and published code samples.

## Out of Scope

- Third-party infrastructure (GitHub, Cloudflare CDN)
- CVE write-ups documenting other vendors' vulnerabilities

## Supported Versions

Only the current `main` branch is maintained.
